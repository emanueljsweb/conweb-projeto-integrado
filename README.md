# Projeto Integrado - Construção de Software para Web

Esse repositório armazena o resultado do nosso projeto integrado na disciplina de Construção de Software para a Web, onde construímos a ideia de uma aplicação para o gerenciamento de atividades complementares. Transformamos algumas das telas produzidas na disciplina de Experiência de Interface com o Usuário em HTML & CSS.
